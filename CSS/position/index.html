CSS Positions

  1. Static:The original or default position of an element.
  2. Relative: Element is positioned relative to its normal/original position. Same as static positioning just that we can move an element by adding offset values..Element stays in normal document form.
  3. Absolute: Element is positioned relative to its parent. if parent is not present then element is positioned relative to the main document.Element is removed from the normal document flow which means that no space is created for that element.
